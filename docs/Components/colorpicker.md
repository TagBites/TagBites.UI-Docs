# Color picker

**Behavior control attribute:**  `UIColorPickerAttribute`

**Types automatically recognized:** `TagBites.UI.UIColor`

**Acceptable types**: `TagBites.UI.UIColor`

### Example
```csharp
[UIColorPicker]
public UIColor MyColor { get; set; }
```
